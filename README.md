# CrossChronox

CrossChronox is a Cross-platform rhythm game based on Rust.
It works on Windows, macOS, and Linux.
We can play bms, bme, bml, pms, and bmson.
Read Wiki for more information.

LICENSE : LGPL v3

(WIP)

## Build

```
cargo build
cargo run
```
